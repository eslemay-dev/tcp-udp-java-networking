# Java TCP & UDP Client-Server

This project demonstrates basic network communication using Java.

## Features

- TCP client-server communication
- UDP client-server communication
- Socket programming
- Message exchange between devices

## Technologies

- Java
- Socket Programming

## How to Run

### Compile
```bash
javac *.java
```

### Run TCP Server
```bash
java TCPServer
```

### Run TCP Client
```bash
java TCPClient
```

### Run UDP Server
```bash
java UDPServer
```

### Run UDP Client
```bash
java UDPClient
```

## Description

This project shows how TCP and UDP protocols work in real-world communication scenarios using Java sockets.
